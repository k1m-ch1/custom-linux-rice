# Software Used

## Core desktop stuff

- Linux Distribution: Arch
- Window Manager & compositor: `hyprland`
- Bar: `waybar`
- App launcher: `rofi`

## Graphics

- theme: Catppuccin mocha
- font: JetBrainsMonoNFM-Regular

## Other programs

- Music player: `ncmpcpp`
- Music visualizer: `cava`
- Prompt: `starship`
- Task manager: `btop`
- Text editor: `nvim`
